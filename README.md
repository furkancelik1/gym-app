🏆 Gönen Fit - Mobil Antrenman Asistanı
Gönen Fit, Gönen yerel topluluğundaki sporcuların antrenmanlarını takip etmeleri, hedeflerini yönetmeleri ve birbirleriyle rekabet etmeleri için tasarlanmış modern bir Fullstack mobil uygulamadır.

✨ Özellikler
Akıllı Antrenman Takibi: Hareket isimlerini otomatik tanıyan ve set/tekrar bazlı kayıt tutan sistem.

Liderlik Tablosu (Gamification): Tamamlanan antrenmanlardan "Fit Puan" kazanma ve diğer sporcularla yarışma.

Kişisel Hedefler: Kilo takibi ve haftalık antrenman hedefi belirleme alanı.

Profil Yönetimi: Kişiselleştirilmiş kullanıcı adı ve güvenli oturum yönetimi (Firebase Auth).

Modern Arayüz: Karanlık mod destekli, kullanıcı dostu tasarım.

🚀 Kullanılan Teknolojiler
Frontend: React Native, Expo Router.

Backend: Firebase Firestore (NoSQL), Firebase Authentication.

Yayın (Web): Render.

Dil: TypeScript / JavaScript.

🛠️ Kurulum
Projeyi yerelinizde çalıştırmak için:

Depoyu klonlayın:

Bash
git clone https://github.com/furkancelik1/gym-app.git
Bağımlılıkları yükleyin:

Bash
npm install
Firebase yapılandırması için .env dosyanızı oluşturun (Environment Variables).

Uygulamayı başlatın:

Bash
npx expo start
